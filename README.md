# Homelab Overview

This repository documents my personal homelab environment, which I use to develop and test skills across virtualisation, networking, remote access, and self-hosted services.

## Environment Overview

- **Virtualisation:** Proxmox host running multiple Linux VMs
- **Containerisation:** Docker for service deployment
- **Networking:** UniFi switches and access points
- **Remote Access:** Tailscale (including subnet routing)
- **Reverse Proxy:** Caddy for HTTPS access to internal services
- **Storage:** NAS for media, backups, and lab data

## Key Services

- Home Assistant (home automation)
- Supporting Docker services
- Internal applications exposed via reverse proxy

## What I Use This For

- Testing new tools and services safely
- Learning networking and infrastructure concepts
- Running self-hosted services
- Practising secure remote access setups

## Notes

This lab is continuously evolving as I test new configurations and improve security and reliability.
